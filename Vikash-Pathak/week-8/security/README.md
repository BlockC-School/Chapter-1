# Basic AccessControl Security Auditing Project with Hardhat & Truffle

Try running some of the following tasks:
## HardHat Cmd
```shell
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat node
node scripts/sample-script.js
npx hardhat help
```
## Truffle Cmd
```shell
truffle 
truffle run
truffle compile
truffle migrate
truffle test
truffle deploy
truffle develop
```
